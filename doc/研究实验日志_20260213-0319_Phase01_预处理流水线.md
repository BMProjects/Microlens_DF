## 背景与物理原理

### 暗场显微镜成像机制

本项目的检测目标基于暗场显微成像的物理原理：入射光以大角度（约 70°）斜射镜片表面，无缺陷区域全反射、不进入相机，成像为全黑背景；缺陷（划痕沟槽、凹坑、密集微裂纹）对入射光产生散射，散射光进入相机，形成暗背景上的亮斑。

```
散射机制与图像表现对照：
┌─────────────────┬──────────────────────────┬─────────────────────────┐
│ 缺陷类型         │ 物理机制                   │ 暗场图像表现              │
├─────────────────┼──────────────────────────┼─────────────────────────┤
│ 细划痕（<5μm宽） │ 沟槽衍射+均匀散射          │ 细长亮线，散射均匀         │
│ 深划痕（>10μm）  │ 沟槽+边缘碎裂散射          │ 宽亮线，边缘不规则         │
│ 凹坑（spot）     │ 局部坑洞散射               │ 孤立亮点，近圆形           │
│ 密集磨损（critical）│ 大面积微裂纹网络散射      │ 区域性高亮，边界模糊       │
└─────────────────┴──────────────────────────┴─────────────────────────┘
```

## 预处理流水线架构

### 系统目标

将原始 4096×3000 暗场灰度图转换为背景归一化、ROI 对齐的标准图，消除：（1）照明不均匀的高光环形区域（Ring）；（2）帧间平移/旋转/轻微缩放偏差；（3）非镜片区域（四角全黑）的干扰。

### 模块架构

```
preprocessing/
├── pipeline.py             主流水线（PreprocessPipeline + 数据类）
├── background_fusion.py    多帧平均 + 离焦模拟
├── roi_builder.py          高光ring检测 + ROI提取
├── registration.py         ring模板配准（相位相关 + ECC仿射）
└── brightness_correction.py 亮度修正 + 最终化
```

各模块职责严格分离，`pipeline.py` 仅负责编排，其余模块为纯函数实现。

## Exp-P1-01：初始方案测试——复杂多帧配准流水线

**假设**：5 帧背景图存在帧间偏移，需要逐帧配准后融合；RANSAC 圆拟合可以精确提取高光 Ring 区域。

**实验设置**：
- 流水线：5 帧背景图 → 弧线提取 → RANSAC 圆拟合 → 帧间 ECC 配准 → 中位数融合 → 背景模板
- 可调参数总数：24 个
- 测试图像：247 张 BMP，4096×3000，uint8 灰度

**发现的问题**：
1. 每帧单独配准引入额外误差——暗场背景本身极稳定，各帧之间差异远小于噪声水平，独立配准是多余步骤
2. RANSAC 圆拟合对低对比度区域（Ring 边缘渐变区）噪声敏感，稳定性差
3. 中位数融合比直接平均在此场景下无统计优势（背景无脉冲噪声）
4. 流程冗余，调试困难，参数耦合度高

**改进决策**：
直接平均 5 帧 → 离焦模糊（σ=30）→ 中心相对阈值 Ring 检测 → ROI 构建

**结果**：参数从 24 个降为 14 个，流程更鲁棒，稳定性显著提升。

## Exp-P1-02：高光 Ring 检测——全局百分位阈值失效分析

**研究问题**：全局 p94 百分位阈值是否适用于非均匀暗场图像的 Ring 提取？

**假设（被否证）**：使用全局 p94 百分位作为 Ring 阈值，能稳定区分 Ring 高亮区和中心均匀区。

**定量验证数据**：

```
背景图像灰度结构（B_avg）：
  中心均一灰度   ~43（ROI 内均值）
  Ring 高光区域   65～91（距镜片中心约 r=1900px 处）
  四角全黑       ~28（非 ROI 区）

方案A（σ=12 离焦模糊后）：
  全图 p94 = 50
  中心区域 p99 = 49
  差距 = 1 个灰度值 → 阈值极不稳定，任意噪声可导致误分类

方案B（σ=30 离焦模糊后）：
  Ring 均值 @ r=1900 px = 65.2
  中心区域 p99 = 49
  差距 = 16 个灰度单位 → 稳定，对亮度漂移有较强容忍性
```

**最终改进方案**：

1. 离焦 σ 从 12 提升到 30（等效：6×σ_per_pass=6 的高斯模糊叠加 5 次），充分平滑 Ring 边缘
2. 阈值改为中心相对阈值：
   `th = center_level × center_ratio`，其中 `center_ratio = 1.25`
   `center_level` = 图像中心 40% 区域的亮度中位数
3. 附加约束：`th ≥ center_level + max(MAD×4.5, center_level×10%)`，防止极端低对比度场景误判
4. Ring 膨胀 25 px：形态学膨胀桥接 Ring 边缘渐变区的细小断缝

**验证结果**（`output/preprocess_test/calibration/`）：

| 指标 | 数值 |
|------|------|
| Ring 像素数 | 1,160,452 |
| ROI 像素数 | 9,897,892（约占图像 80%）|
| 中心亮度 median | 44.41 |
| 中心亮度 MAD | 1.46 |
| Ring/中心对比度 | ≈1.47×（稳定触发阈值）|

**结论**：中心相对阈值策略在不同光照条件下具有自适应性，无需手动调参。

## Exp-P1-03：ECC 图像配准失效分析——32 张问题图像的定位与修复

**研究问题**：哪些图像的 ECC 配准会失效？失效的定量判据和根因是什么？

**问题发现（人工目视检查 247 张结果）**：
32 张图像（包括 1l, 1r, 13l, 13r, 14l 等）的右半边高光环形区域匹配错误，表现为：模板覆盖目标图像 Ring 区域的外围边缘，而非对齐中心；右侧 Ring 高亮完全未被模板覆盖。

**配准流程（两阶段）**：

```
目标图像（raw）
    │
    ▼ 相位相关（Phase Correlation）—— 快速估计粗平移量（±200px 范围内）
  初始平移估计 (dx₀, dy₀)
    │
    ▼ ECC 仿射细化（Enhanced Correlation Coefficient）
      MOTION_AFFINE 模型：平移 + 旋转 + 缩放（限制 ±5%）
      最大迭代 200 次，收敛阈值 1e-6
  warp_matrix (2×3)
    │
    ▼ 仿射变换 → aligned → 背景减除 → 最终化
```

**根因分析**：

1. 这批 32 张图像的 Ring 区域实际平移量大（dy > 50 px），超出相位相关的可靠估计范围（约 ±30 px）
2. 粗估计偏差导致 ECC 初始状态偏离真实对齐，陷入局部最优
3. 局部最优对应一个错误的仿射变换（ECC score < 0.6），使 Ring 模板偏向图像边缘

**解决方案：级联回退策略**

```
策略 1：仿射 ECC（MOTION_AFFINE，scale ≤ 5%）
    → 成功（score ≥ 0.75 且 scale 偏差 ≤ 5%）：采用
    → 失败：降级到策略 2
策略 2：欧氏 ECC（MOTION_EUCLIDEAN，仅旋转+平移，锁定缩放=1.0）
    → 成功（score ≥ 0.75）：采用
    → 失败：降级到策略 3
策略 3：纯平移（相位相关结果，无细化）
    → 总是采用（保底）
```

**量化验证结果**：

| 指标 | 修复前 | 修复后 |
|------|--------|--------|
| 问题图像数量 | 32 张 | 0 张（全部修复）|
| ECC score 均值（32 张中） | < 0.60 | > 0.92（3 张极端异常除外）|
| 极端异常图像 | — | 103l: 0.613, 120r: 0.571, 91l: 0.449 |

**注**：3 张极端异常图像（ECC < 0.65）源于拍摄时曝光/焦距异常（非配准算法问题），已记录，不排除训练集。
