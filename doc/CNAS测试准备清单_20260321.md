## 一、测试前必须完成的工作

### 1.1 独立测试集准备（最高优先级）

当前 247 张图像全部参与训练或验证，**缺乏第三方可信的独立测试集**。

| 步骤 | 操作 | 说明 |
|------|------|------|
| ① 保留测试集 | 从 247 张中挑选 **20-30 张** 作为独立测试集 | 应覆盖各类缺陷程度（轻微/中度/严重）|
| ② 避免信息泄露 | 测试集图像不能出现在 train/val 切片中 | 需重新划分数据集 |
| ③ 标注测试集 | 对测试集进行人工复核标注 | 不能依赖经典检测器自动标注 |
| ④ 固定测试集 | 测试集一旦确定不再改动 | 记录文件名和 MD5 哈希 |

> **临时方案（时间紧迫时）**：当前 val 集（2,150 tiles，50张原图）可作为半独立测试集。
> Stage 2 训练中 val 集仅用于早停监控，权重选择基于最优 epoch，无针对 val 集的参数调整。
> 需在报告中明确说明划分方式，提供 `tile_index.csv` 作为证明。

## 二、技术报告框架（供撰写参考）

```
暗场显微镜镜片缺陷智能检测系统 — 中期技术报告
版本：v1.0    日期：2026-03

1. 项目概述
   1.1 检测目标：暗场图像中的划痕(scratch)、斑点(spot)、严重缺陷区(critical)
   1.2 系统架构：预处理 → 切片 → YOLOv12m 检测 → 结果聚合
   1.3 中期测试目标：mAP@0.5 ≥ 60%

2. 数据集
   2.1 图像规格：4096×3000px，uint8 灰度，暗场显微镜采集
   2.2 标注规模：247张原图 → 10,621 tiles（640×640）
   2.3 类别分布：scratch 68% / spot 17% / critical 15%
   2.4 标注质量优化：自动清洗（去除噪声框13.6%）+ 伪标签补漏

3. 算法描述
   3.1 图像预处理：背景平均→离焦模板→高光Ring→ECC配准→背景减除
   3.2 切片策略：640×640，stride=560，overlap=80px
   3.3 检测模型：YOLOv12m（ultralytics 8.4.24）
       - 输入：灰度→RGB复制（3通道）
       - 损失：Varifocal Loss + DFL，cls权重=1.0
       - 增强：关闭色调/饱和度，保留亮度抖动，180°旋转
   3.4 两阶段训练：基线训练 + 标注清洗重训练

4. 实验结果
   4.1 阶段对比：Stage1 mAP=0.5257 → Stage2 mAP=0.6621（+25.8%）
   4.2 各类 AP（待补充）
   4.3 典型正确检测示例
   4.4 典型失败案例分析

5. 误差分析
   5.1 主要漏检场景：密集划痕区（经典检测器过标注）
   5.2 误检场景：亮度波动被检测为 spot
   5.3 跨切片划痕的局限性说明

6. 结论
   当前系统在 val 集上 mAP@0.5 = 0.6621，达到中期目标 ≥ 60%。
```
