## 一、暗场显微镜成像物理原理

### 1.1 暗场成像与明场成像的本质区别

**明场成像（Bright Field）**：光源正面照射样品，光线直接进入物镜，背景为明亮均一的白场，缺陷表现为暗色轮廓。

**暗场成像（Dark Field）**：光源以大角度（通常 > 60°）侧面或斜面照射，无缺陷的平滑镜片表面产生全反射，进入物镜的光极少 → 全暗背景；当镜片表面存在划痕、凹坑、微裂纹等缺陷时，缺陷边缘和沟槽将入射光散射至各方向，部分散射光进入物镜 → 缺陷在暗背景下呈现高亮响应。

**信噪比优势**：暗场成像的信噪比（SNR）远高于明场，细微划痕在明场下几乎不可见，而在暗场下表现为清晰亮线。这是该技术被选用于工业镜片检测的核心原因。

### 1.2 本项目成像系统的物理特性

本项目使用的暗场显微镜系统具有以下物理特性：

| 参数 | 值 | 物理意义 |
|------|-----|---------|
| 图像分辨率 | 4096×3000 px | 高分辨率，可分辨 1-2 px 宽的细划痕 |
| 图像格式 | uint8 灰度（0-255） | 单通道强度图像 |
| 背景图像 | 5帧均值 | 消除热噪声，获得稳定背景模板 |
| 高光Ring | 镜片边缘高光环 | 镜片边框散射引起，非缺陷，需要过滤 |
| ROI | 圆形镜片有效区域 | 直径约 3000px，需精确定位 |
| 中心亮度 | ~43（uint8） | 暗场基准，无缺陷区域的背景灰度 |
| 缺陷灰度 | 65-255 | 散射光强度，与缺陷深度/宽度正相关 |

### 1.3 缺陷的散射机理与分类

不同缺陷类型对应不同的散射机理，在暗场图像中呈现不同的视觉特征：

| 缺陷类型 | 物理机理 | 暗场图像特征 | YOLO类别 |
|---------|---------|------------|---------|
| 细划痕（细线）| 沟槽状镜面损伤，沟槽侧壁散射 | 细长亮线，宽度1-5px，长宽比>5 | scratch |
| 粗划痕（粗线）| 沟槽+边缘碎裂散射 | 宽亮线，边缘不规则，宽度5-30px | scratch |
| 凹坑（spot）| 局部点状坑洞，全向散射 | 孤立亮点，近圆形，直径5-30px | spot |
| 密集磨损 | 大面积微裂纹网络，弥散散射 | 区域性高亮，边界模糊 | critical |
| 大面积缺损 | 镜片局部破损，面积>0.1mm² | 不规则大面积亮区 | critical |

### 1.4 成像噪声来源分析

暗场成像的噪声来源与典型图像中的表现：

1. **背景不均匀性**：即使无缺陷，镜片中心区域因本底散射呈现约5个灰度单位的随机波动。由背景图模板（5帧均值+离焦模糊）进行除法校正后，本底波动降至约2单位。

2. **灰尘粒子**：镜片表面或光路中的灰尘粒子在暗场下与凹坑（spot）视觉上几乎无法区分。这是spot类别存在误标的主要来源。

3. **照明非均匀性**：暗场照明在ROI内的光强分布呈轻微梯度（边缘略亮于中心）。背景除法校正可消除大部分非均匀性，但小尺度残留仍影响阈值选取。

4. **Ring高光区域**：镜片边缘框架产生的高亮环形区域，灰度65-91，是误检的主要来源之一。通过ROI掩膜过滤。

## 二、缺陷检测技术路线综合调研

### 2.1 传统图像处理方法

#### 2.1.1 Frangi滤波器（血管/线状结构增强）

Frangi滤波器（Frangi et al., 1998）基于Hessian矩阵的二阶导数计算图像中"管状/线状结构"的响应。

**数学原理**：
对每个像素 $(x,y)$，计算Hessian矩阵 $H = \begin{bmatrix} I_{xx} & I_{xy} \\ I_{xy} & I_{yy} \end{bmatrix}$ 的两个特征值 $\lambda_1 \leq \lambda_2$。

线状结构判据：
```
R_B = λ1 / λ2     → 接近0表示线状结构（λ1≈0, λ2大）
S = sqrt(λ1² + λ2²)  → 表示结构强度
Frangi(x,y) = exp(-R_B²/(2β²)) × (1 - exp(-S²/(2c²)))
```

**在本项目中的应用**：
- 多尺度（σ=1,2,4,8px）Frangi滤波，覆盖不同宽度的划痕
- Frangi响应作为标注验证工具：Frangi响应<0.05的"scratch框"很可能是噪声
- 统计发现：44.3%的GT scratch框具有低Frangi响应（验证标注噪声严重）

**局限性**：
- 对实心亮块（damage/crash类型）响应极低（Frangi是脊线检测器，不是面积检测器）
- 在噪声较高的区域（gamma=0.4增强后），Frangi会将噪声纹理误识别为线状结构

#### 2.1.2 Top-hat变换（背景分离）

形态学Top-hat变换 $T(f) = f - f \circ B$（其中 $\circ$ 为开运算，$B$ 为结构元素）。

在本项目中用于提取比背景（由开运算估计的平滑背景）更亮的局部特征，核大小15px覆盖宽度<15px的缺陷。与Frangi滤波的差异：Top-hat对面积较小的亮区（spot类型）响应更好。

#### 2.1.3 Otsu阈值分割

Otsu算法（Otsu, 1979）自动选取使类间方差最大的阈值，将图像分割为前景（缺陷）和背景。

在本项目的切片标注生成中：
- 对每个640×640切片的候选图（Frangi+TopHat+亮度融合）进行Otsu阈值
- `otsu_floor=0.08`防止纯暗切片阈值过低（设置阈值下限）

**局限性**：全局Otsu对局部背景不均匀的图像效果较差，但在背景校正后的切片上效果可接受。

### 2.2 深度学习目标检测方法

#### 2.2.1 YOLO系列发展脉络

| 版本 | 关键技术贡献 | 适用场景 |
|------|------------|---------|
| YOLOv1-v3 | 统一网络单阶段检测框架 | 速度优先 |
| YOLOv5/v8 | anchor-free，CSP骨干，多尺度FPN | 通用检测 |
| YOLOv8-OBB | 定向边界框支持 | 旋转目标（文本、卫星图） |
| **YOLOv12m** | **Area Attention内置，区域注意力机制** | **密集小目标，纹理缺陷** |

**选型依据**：YOLOv12m的Area Attention特别适合本项目：
- 划痕在局部区域内具有高度方向相关性（同一方向的相邻像素高度相关）
- Area Attention通过对局部矩形区域计算注意力，有效捕捉这种各向异性特征
- 在缺陷检测基准（MVTec-YOLO等）上优于同规模的YOLOv8m

#### 2.2.2 小目标检测技术综述

本项目的缺陷目标具有典型的小目标特征：spot类别中位数仅19px，在640×640切片中占比仅 (19/640)² ≈ 0.09%。相关技术：

**多尺度特征融合（FPN/PANet）**：
- FPN (Feature Pyramid Network, Lin et al. 2017)：将高层语义特征向下传递至高分辨率特征图
- PANet：在FPN基础上增加自下而上的路径，强化位置信息传递
- YOLOv12m的neck部分集成了PANet变体，为本项目提供多尺度特征

**SAHI（Slicing Aided Hyper Inference）**（Akyon et al., 2022）：
- 将大图切分为重叠子图分别推理，然后合并结果
- 专为小目标检测设计，在VisDrone等小目标基准上提升约15-20% mAP
- 本项目中的应用：4096×3000原图 → 640×640滑窗切片（stride=480），每张图约113个切片

#### 2.2.3 细长目标检测的特殊挑战

划痕（scratch）属于"超高长宽比细长目标"（长宽比通常 > 5，最高可达 50:1）。标准YOLO检测框架面临的挑战：

**IoU计算失效**：
```
假设真实划痕框：200×8px（面积1600）
预测框水平偏移4px：196×8px（面积1568）
实际IoU = 交集面积/并集面积 = 1568/(1600+1600-1568) = 0.980（看起来很高）

但若划痕是垂直的，预测框垂直偏移4px：
真实: x=[0,8], y=[0,200]
预测: x=[0,8], y=[4,204]
IoU = 196*8 / (200*8 + 204*8 - 196*8) = 1568/1664 = 0.942

若预测框比真实框短50px（忽略了划痕末端）：
真实: 200×8=1600，预测: 150×8=1200
IoU = 1200/1600 = 0.75 → 只有0.75的IoU，按IoU@0.5算算正样本，但定位差

若换为IoU@0.75，则IoU<0.75 → 算FP！
```

这解释了为什么本项目mAP@0.5（0.6765）与mAP@0.5:0.95（0.4541）差距达0.22。

**定向边界框（OBB）的潜在价值**：
- 对于斜向划痕，轴对齐bbox面积 ≫ 缺陷真实面积，导致IoU天然偏低
- OBB（旋转框）可以紧贴划痕方向，显著提高IoU
- 代价：需要重新标注（角度参数），工作量约为原bbox标注的2倍
- 预期收益：mAP@0.5:0.95提升3-5%

### 2.3 工业缺陷检测数据集与基准

#### 2.3.1 MVTec AD数据集

MVTec Anomaly Detection（Bergmann et al., 2019）是工业缺陷检测最常用的基准数据集。包含15种工业产品、5354张图像、73种缺陷类型。

与本项目的关键差异：
- MVTec：多类别工业产品，缺陷清晰，有正常/异常标签
- 本项目：单一场景（镜片暗场），缺陷连续分布（无"正常"图像），高分辨率

#### 2.3.2 VisDrone数据集（无人机小目标）

VisDrone是小目标检测基准，图像尺寸2000×1500+，目标尺寸通常<20px，与本项目的spot类别（中位数19px）特征相近。SAHI在VisDrone上可提升约15% mAP，这为本项目采用SAHI方案提供了参考依据。

#### 2.3.3 相关光学缺陷检测工作

| 工作 | 方法 | 指标 | 与本项目关联 |
|------|------|------|------------|
| PCB缺陷检测（DeepPCB） | Faster RCNN | AP≈0.90 | PCB缺陷无散射特性，对比度高于暗场 |
| 太阳能电池EL缺陷 | U-Net分割 | Dice≈0.85 | 相似的高分辨率工业图像 |
| 晶圆表面缺陷 | 半监督YOLO | mAP≈0.72 | 与本项目最接近（暗场、小缺陷、半标注）|
| 光学玻璃划痕 | Frangi+CNN | Precision≈0.85 | 相同缺陷类型，不同成像模式 |

## 三、标注策略与半监督学习技术综述

### 3.1 弱监督与半自动标注

#### 3.1.1 传统算法初标 + 人工审核（本项目采用）

**流程**：传统图像处理（Frangi+TopHat+Otsu）→ 自动生成初始标注 → 人工审核修正

**优势**：
- 无需任何先验标注即可产生海量初始标注（247张图→112,929框）
- 标注成本远低于纯人工标注

**问题（已量化）**：
- 噪声框率：13.6%（15,338框）
- scratch类Frangi弱响应率：44.3%（存疑框）
- 漏标率（估计）：10-20%（经伪标签验证补充8,613框）

#### 3.1.2 主动学习（Active Learning）

主动学习框架：模型对"不确定样本"请求人工标注，使标注成本最小化。

**与本项目的关联**：`scripts/audit_stage2_labels.py`实现了可疑度排序：
```
suspicion = FP数 + FN数 + 2×类别错误数
```
这本质上是一种基于模型不确定性的主动学习样本选择策略。

#### 3.1.3 伪标签（Pseudo Labels）

核心思想（Lee, 2013）：用当前模型对无标注数据推理，将高置信度预测作为伪标签加入训练集。

**本项目的改进版本**：
- 多策略推理（6组）代替单次推理，通过一致性筛选过滤低质量伪标签
- 仅将与GT不重叠（IoU<0.5）的高置信预测作为"补漏"标注（非替换GT）
- 效果：添加8,613框（+11.4%），Stage2 recall从0.50→0.66

### 3.2 标注噪声鲁棒训练

Focal Loss（Lin et al., 2017）原本为解决one-stage检测器的前景/背景类别不均衡问题设计，同时对噪声标签具有一定的鲁棒性（难分样本权重高，但噪声通常是"难以置信的简单或困难样本"，权重低）。

本项目使用 `fl_gamma=2.0`：
- 对 p=0.5 的预测（中等置信度）保持接近正常权重
- 对 p≈1.0 的预测（过拟合标志）降低权重
- 间接减少了噪声标注对训练的影响

## 四、磨损评分系统的视光学基础

### 4.1 镜片磨损对人眼视觉的影响机理

眼镜镜片是精密光学元件，表面磨损通过以下机理影响佩戴者视觉：

**散射导致的对比度下降（Haze）**：
- 密集细划痕形成散射网络，将部分透过光散射至非成像方向
- 等效于在视场中叠加一个均匀散射光背景，降低视觉对比度
- 在低对比度场景（如雾天、夜晚）下最为明显
- 量化指标：Haze Index（与划痕密度×交叉点密度正相关）

**眩光（Glare）**：
- 深划痕和大面积缺损将高亮光源（车灯、路灯）散射为放射状光晕
- 核心视区（光学中心±10-15mm）的深划痕在夜间驾驶时产生严重眩光
- 量化指标：Glare Index（∝ Σ划痕长度×散射强度 × 中心区权重）

**局灶性视觉失真**：
- 较大的凹坑（spot，>0.5mm²）可在成像面上产生局部折射异常
- 密集缺损区（critical）可能造成局部焦距偏差

### 4.2 视区分级权重设计依据

镜片视区分级基于人眼视觉生理学原理：

| 视区 | 定义 | 权重依据 |
|------|------|---------|
| 核心视区 | 光学中心半径 ≤40%（≈ROI半径40%） | 正视眼注视时主要使用，缺陷直接影响主视力 |
| 过渡视区 | 40%-75%半径 | 周边视觉使用，影响余光感知和视场宽度 |
| 边缘视区 | >75%半径 | 极偶尔使用，主要影响外观，对功能性视觉影响轻微 |

**半径阈值设计**（本项目：0.40 / 0.75）：
- 40% ROI半径对应镜片"有效光学区"内缘（典型眼镜镜片光学区直径约为镜片直径的75-80%）
- 本项目ROI约3000px直径，40%≈600px，对应约15mm（假设镜片直径约37mm）

### 4.3 WearScore公式设计的理论依据

**对数饱和函数的必要性**：
- 线性函数：单一极端缺陷（如1条200px长划痕）会将评分推满
- 对数饱和：`sat_log(x) = min(scale × log1p(x), 100)` 反映了真实视觉影响的边际递减规律
- 生理学依据：人眼的主观感知强度与刺激强度呈对数关系（Weber-Fechner定律）

**5分量权重的独立性验证**：
- 中心区缺陷密度（f_center）：独立于总缺陷量，捕捉"集中在关键位置"的损伤
- 全局密度（f_density）：与f_center互补，捕捉弥漫性磨损
- critical占比（f_critical）：与密度独立，捕捉"虽少但严重"的大面积缺损
- 散射强度（f_scatter）：代理深度信息，与数量/面积指标互补
- 总量（f_total）：作为全局兜底指标

## 五、项目关键技术决策与方案选型记录

### 5.1 为何选择YOLOv12m而非分割模型

原始规划（《划痕识别整体规划》模块二）提出两阶段方案：
1. Frangi/Hessian候选图生成
2. UNet/SegFormer精细分割

**放弃分割方案的原因**：
- 标注成本：像素级掩码标注需要专业人员，成本约为bbox的5-10倍
- 暗场边界不清晰：缺陷散射光的物理边界是渐变的，像素级GT本身具有±5px的固有不确定性
- 工程复杂度：YOLOv12m端到端训练，无需两阶段协调
- 实际效果：bbox检测+SAHI推理端连接在达到60%mAP目标上已足够

**bbox vs mask的核心权衡**：
```
bbox优势：
  - 标注成本低（自动生成可行）
  - 训练稳定，无需像素级GT
  - 推理速度快（无掩码解码）

mask优势：
  - 划痕真实长度/形态（bbox为近似）
  - OBB检测可通过mask回投获得
  - 为WearMetrics提供精确骨架
```

**结论**：中期目标（60%mAP）阶段选bbox方案性价比最高。长期目标（精确WearMetrics）需要mask支持，可通过SAM（Segment Anything Model）从bbox自动生成，无需重新标注。

### 5.2 为何选择SAHI而非全图检测

**全图检测的问题**：
1. 内存：4096×3000图像在YOLOv12m中需要 ~8GB GPU显存（单张，batch=1）
2. 小目标分辨率：4096×3000 downsampled to 640×640 → 单个缺陷仅3-5px
3. 划痕碎片问题在全图检测中同样存在（模型在训练切片上学习，推理时分辨率不匹配）

**SAHI方案的参数选择**：
```
tile_size = 640px    # 与训练分辨率完全一致
stride = 480px       # overlap = 160px = 25%
# 25% overlap确保：
# - 跨切片划痕在至少1个切片中有足够长度被检出
# - 推理切片总数：≈(4096/480)×(3000/480) ≈ 9×7 = 63张/图（可接受）
```

### 5.3 三阶段训练策略的设计逻辑

```
Stage1（基线）: yolo12m.pt → 半自动标注原始数据 → 建立基线，暴露标注问题
    ↓ 发现：train/val gap=0.73，过拟合，噪声标注是主因
Stage2（清洗+伪标签）: Stage1 best.pt → 清洗标注+伪标签 → mAP 0.52→0.66
    ↓ 发现：gap降至0.04，泛化良好，scratch AP=0.41是瓶颈
Stage3 = Stage2（最优，回滚）: 在Stage2基础上进行人工审核后再训练（Phase 2C）
    实际：mAP=0.6621（val），0.6765（独立测试集）
Stage4（失败实验）: Stage3 best.pt → GT连接重标注 → mAP 0.65→0.47（-27.7%）
    教训：GT连接破坏训练信号
```

微调（Fine-tuning）vs 从头训练（Training from scratch）的选择：
- Stage1 → Stage2采用微调：保留已学特征，仅调整标注引起的偏差
- lr0从0.01降至0.002（1/5）：避免破坏Stage1已学的语义特征
- 实验验证：微调从Stage1的0.52起步，到Stage2的0.66，证明微调有效

## 六、技术引用与参考文献

以下是本项目关键技术的来源文献（按重要性排序）：

### 核心技术

1. **YOLOv12**: Tian et al., "YOLOv12: Attention-Centric Real-Time Object Detectors" (2025). YOLOv12m集成了Area Attention机制，对局部纹理相关目标（如划痕）具有天然优势。

2. **Ultralytics YOLO框架**: https://github.com/ultralytics/ultralytics (v8.4.24). 提供从训练到部署的完整工具链，包括DFL loss、Varifocal loss等先进训练技术。

3. **SAHI**: Akyon et al., "Slicing Aided Hyper Inference and Fine-tuning for Small Object Detection" (ICIP 2022). 专为大图小目标检测设计的推理框架，无需修改模型结构。

4. **Frangi滤波器**: Frangi et al., "Multiscale vessel enhancement filtering" (MICCAI 1998). 多尺度Hessian特征提取线状/管状结构，广泛用于医学图像血管增强，本项目用于划痕候选图生成和标注验证。

5. **Focal Loss**: Lin et al., "Focal Loss for Dense Object Detection" (ICCV 2017). 解决单阶段检测器正负样本极端不均衡问题，本项目fl_gamma=2.0用于缓解background主导的梯度问题。

### 数据增强与标注

6. **伪标签**: Lee, "Pseudo-label: The simple and efficient semi-supervised learning method for deep neural networks" (ICML Workshop 2013). 本项目多策略伪标签方案的理论基础。

7. **Copy-Paste增强**: Ghiasi et al., "Simple Copy-Paste is a Strong Data Augmentation Method for Instance Segmentation" (CVPR 2021). 用于damage类过采样（damage_augment.py）。

8. **ECC图像配准**: Evangelidis & Psarakis, "Parametric image alignment using enhanced correlation coefficient maximization" (TPAMI 2008). 用于背景模板配准（preprocessing/registration.py）。

### 评估方法

9. **COCO AP指标体系**: Lin et al., "Microsoft COCO: Common Objects in Context" (ECCV 2014). 本项目采用mAP@0.5和mAP@0.5:0.95作为主要评估指标。

10. **Otsu阈值**: Otsu, "A threshold selection method from gray-level histograms" (IEEE Trans. SMC 1979). 用于切片标注自动生成的阈值分割步骤。

### 架构参考

11. **FPN**: Lin et al., "Feature Pyramid Networks for Object Detection" (CVPR 2017). YOLOv12m neck中多尺度特征融合的基础。

12. **PANet**: Liu et al., "Path Aggregation Network for Instance Segmentation" (CVPR 2018). YOLOv12m在FPN基础上增加的自下而上路径增强位置信息。

## 七、领域知识：眼镜镜片的标准化检测要求

### 7.1 国际标准概况

- **ISO 8980-1**: 眼镜片单光未割边镜片要求（包含外观要求，但未明确划痕容差的量化标准）
- **GB 10810.1-2014**: 眼镜镜片第1部分（中国国标），规定镜片"有效区内不应有影响视力的疵点"
- **ANSI Z80.1**: 美国眼镜镜片标准

**关键观察**：现行标准对"划痕容差"的描述多为定性（"不影响视力"），缺乏量化阈值。本项目的WearScore体系（A/B/C/D分级）是对这一标准化缺口的填补尝试。

### 7.2 工业检测中的常见实践

| 检测方式 | 优点 | 缺点 | 与本项目的关系 |
|---------|------|------|--------------|
| 人工目检（白光背景）| 简单直接 | 主观，易疲劳，漏检率高 | 本项目的替代目标 |
| 干涉仪（Newton环）| 精确定量（nm级表面形貌） | 价格昂贵，操作复杂，速度慢 | 提供参考精度基准 |
| 暗场散射成像（本项目）| 高灵敏度，高分辨率，可自动化 | 需要光路设计，图像处理复杂 | 核心技术路线 |
| 机器视觉（白光+CNN）| 成熟商业化 | 对暗场特有细划痕不灵敏 | 对比基线 |

### 7.3 镜片寿命与磨损速率参考数据

- 典型树脂镜片（1.67折射率，未镀AR膜）：日常使用6个月后出现划痕的概率约70%（根据文献估计）
- 镀硬化膜镜片：寿命约延长2-3倍
- 运动使用/暴力清洁：加速磨损约5-10倍
- 本项目数据集中：247张镜片覆盖"轻度~极重度"全损伤谱系，反映真实使用场景分布
