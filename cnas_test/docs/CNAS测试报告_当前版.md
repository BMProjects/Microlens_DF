## 3. 指标计算方法

1. 固定留出测试数据集清单位于 [`test_set_v1.json`](/home/bm/Dev/Microlens_DF/cnas_test/manifests/test_set_v1.json)。
2. 按图像 stem 收集对应 `860` 个测试切片和真实标注框。
3. 对全部测试切片进行推理，得到预测框、类别和置信度。
4. 若预测框与真实框类别一致且 `IoU >= 0.5`，记为 `TP`；未匹配预测记为 `FP`；未匹配真实记为 `FN`。
5. `Precision = TP / (TP + FP)`。
6. `Recall = TP / (TP + FN)`。
7. 单类 `AP@0.5` 为 `Precision-Recall` 曲线下面积。
8. `mAP@0.5 = (AP50_scratch + AP50_spot + AP50_critical) / 3`。
9. 当 `mAP@0.5 >= 0.60` 时，判定测试通过。
