## 关系说明

- `cnas_test/docs/`：结构化 Markdown 源，便于版本管理和后续改写
- `doc/`：对外提交或归档时使用的正式文档目录
- `cnas_test/outputs/latest/reports/`：每次实际执行评测后自动生成的测试结果报告
